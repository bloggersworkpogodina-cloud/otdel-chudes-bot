# Хранитель чудес — Telegram-бот для чата

Функции:
- удаляет системное уведомление о вступлении в чат;
- приветствует нового участника в чате «Личный отдел чудес»;
- показывает кнопку «📜 Правила чата»;
- правила видит только тот, кто нажал кнопку;
- приветствие удаляется автоматически через 10 минут.

## Как запустить

1. Установить зависимости:
```bash
pip install -r requirements.txt
```

2. Создать файл `.env` по примеру `.env.example` и вставить новый токен:
```bash
BOT_TOKEN=123456:ABC...
DELETE_WELCOME_AFTER_SECONDS=600
```

3. Запустить бота:
```bash
python bot.py
```

## Важно в Telegram

Бота нужно добавить в чат администратором и выдать право:
- удалять сообщения.

Иначе он не сможет удалять системные уведомления о вступлении.

# Хранитель чудес — Mini App

Telegram-бот для чата «Личный отдел чудес».

Функции:
- удаляет системное уведомление о вступлении в чат;
- приветствует нового участника и отмечает его кликабельным именем;
- показывает кнопку «📜 Правила чата»;
- открывает правила в отдельном окне Telegram Mini App;
- приветствие автоматически удаляется через 10 минут.

## Файлы

- `bot.py` — код бота и мини-страницы правил;
- `requirements.txt` — зависимости;
- `Procfile` — команда запуска Railway;
- `runtime.txt` — фиксирует Python 3.12 для стабильной сборки.

## Railway variables

Обязательная переменная:
- `BOT_TOKEN` — токен бота из BotFather.

Дополнительные переменные:
- `WEBAPP_URL` — публичный URL Railway-сервиса. Нужен для открытия правил в отдельном окне.
- `WELCOME_DELETE_SECONDS` — через сколько секунд удалять приветствие. По умолчанию 600.

## После деплоя

1. В Railway открыть сервис `otdel-chudes-bot`.
2. В Settings / Networking сгенерировать публичный домен.
3. Скопировать URL домена.
4. В Variables добавить/обновить `WEBAPP_URL` этим URL.
5. Дождаться redeploy.
6. В Telegram добавить бота администратором в чат с правом удалять сообщения.

# Хранитель чудес

Telegram-бот для чата «Личный отдел чудес».

Что делает:
- удаляет системные сообщения о вступлении в чат;
- приветствует нового участника с кликабельным тегом;
- показывает кнопку «📜 Правила чата»;
- открывает правила в отдельном Telegram-окне через Web App/Mini App;
- автоматически удаляет приветствие через 10 минут.

Railway variables:
- BOT_TOKEN — токен бота из BotFather.
- WEBAPP_URL — ссылка на Railway-домен приложения, например https://xxx.up.railway.app/rules
- WELCOME_DELETE_SECONDS — можно не указывать, по умолчанию 600.

Важно:
1. Бот должен быть администратором чата.
2. У бота должно быть право удалять сообщения.
3. После деплоя в Railway нужно открыть сервис → Settings → Networking → Generate Domain.
4. Скопировать домен и добавить переменную WEBAPP_URL со значением: https://домен/rules
